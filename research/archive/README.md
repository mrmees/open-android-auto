This directory contains copied research materials from related local repositories.

- `docs/OpenAutoPro_archive_information/firmware/`: vendor firmware notes

- `analysis/tools/apk_indexer/`: reproducible APK indexing toolchain
- `analysis-projection/.../apk-index/`: generated JSON/report outputs for AA APK v16.1
- `captures/`: selected capture findings and timeline summaries
